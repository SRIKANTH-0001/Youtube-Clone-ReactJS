# YouTube Clone

A small React-based YouTube-style frontend built with Vite. This project demonstrates a simple video browsing UI, sidebar navigation, video playback page, and recommended videos component.

**Quick Preview**
- **Live dev:** `npm run dev`
- **Build:** `npm run build`
- **Preview build:** `npm run preview`

**Features**
- Browse a feed of videos (mock data)
- Sidebar navigation
- Video playback page with recommended videos
- Responsive layout scaffolding

**Tech Stack**
- React (via Vite)
- Vite dev server and build
- CSS modules / plain CSS for components

**Getting Started**
1. Install dependencies:

```
npm install
```

2. Run the development server:

```
npm run dev
```

3. Build for production:

```
npm run build
```

4. Preview the production build locally:

```
npm run preview
```

**Project Structure (key files)**
- `src/main.jsx` — App bootstrap and router
- `src/App.jsx` — Main app layout
- `src/Components/Feed` — Feed UI
- `src/Components/Sidebar` — Sidebar navigation
- `src/Components/PlayVideo` — Video playback view

**Development Notes**
- Video data is currently mocked in `src/data.js`.
- Routing is handled with `react-router-dom` (see `Pages/Home` and `Pages/Video`).
- Add/replace data fetching logic to connect a real API.
